# 🎵 Music Manager SaaS

A full-stack SaaS platform to manage and convert playlists across platforms like Spotify and YouTube.

## 🚀 Features (in progress)
- Spotify OAuth authentication
- User authentication (JWT)
- RBAC authorization
- Projects & Tasks system
- File uploads
- Pagination, filtering, search

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Auth
- Multer
- Spotify API

## 📌 Upcoming
- Spotify playlist fetching
- Playlist conversion to YouTube
- AI-powered music assistant
